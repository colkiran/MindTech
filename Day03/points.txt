Pytest
------
Pytest is a test framework in python, it is mainly used to write API test cases.

pip install pytest

1. we can run multiple test cases parallelly that reduces the execution time

2. pytest has its own way to detect the test file or test functions automatically

3. pytest will skip a subset of the test during execution

4. pytest allows us to run a subset of the entire test suite

5. pytest is an opensource
